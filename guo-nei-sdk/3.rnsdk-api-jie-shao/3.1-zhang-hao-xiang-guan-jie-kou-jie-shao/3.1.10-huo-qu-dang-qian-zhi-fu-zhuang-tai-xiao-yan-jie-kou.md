# 3.1.10 获取当前支付状态校验接口

#### 功能说明：

 研发方可以调用该接口在下单前查看用户的支付状态，查看是否可以下单。若有不详请联系SDK服务器端技术人员。

#### 接口形式：

```objectivec
-(void)getPayStatusWithCompletionHandler:(RNQueryHandler)handler;
```

#### 接口参数：

 handler：回调，取result.data 中的json字符串，根据需求自行解析相应字段。

#### 接口示例：

```objectivec
[[RNSDKApi sharedInstance]getPayStatusWithCompletionHandler:^(int code, NSString *msg, RNQueryResponse *result) {
         if (code == RN_RESPONSE_SUCCESS_CODE) {
               //请求成功，具体的状态码在result.data中，若不详请联系SDK服务器端技术人员
                   NSLog(@"查询支付状态请求成功，data %@ ",result.data);
               }else{
                   //失败
                   NSLog(@"失败，code %d ,msg%@",code,msg);
               }
    }];
```
