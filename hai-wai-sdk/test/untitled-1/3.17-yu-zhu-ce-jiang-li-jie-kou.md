# 3.17 预注册奖励接口

#### 功能说明： 

用户在苹果商店预注册后，首次下载并打开游戏后，给予的游戏奖励。该接口不会触发支付操作，在沙盒测试阶段，首次会弹出Apple ID登录框。在上线后正式环境下不会触发Apple ID登录框。仅限新游戏首次上线时调用，老游戏无需调用该接口。

#### 接口形式：

```objectivec
+(void)preorderRewardProductId:(NSString)productId roleId:(NSString)roleId serverId:(NSString)serverId extraData:( NSString )extraData completionHandler:(R2BaseCompletionHandler)handler; 
```

#### 参数说明：

 productId: 奖励产品product id,可以为iTunes Connect后台配置的内购产品中任选一个奖励产品，无需单独注册奖励产品。

 roleId: 当前角色id

serverId: 当前服务器ID标识

 extraData: 额外数据，透传字段，可以使用该字段来透传一些额外数据

 handler：回调 

#### 接口示例：

```objectivec
 [R2SDKApi preorderRewardProductId:@"product id" roleId:@"role id" serverId:@"server id" extraData:@"extra data" completionHandler:^(int code, NSString *msg, R2SDKBaseResponse *result) {
        
    }];

```
